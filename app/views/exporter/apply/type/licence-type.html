Which export licence do you want to apply for?
Standard Licences are specific to the company and the recipient (consignee). They are for a set quantity and set value of goods. You will need to provide support documentation with your application.
Open Licences cover long-term projects and repeat business. This is company specific, with no set quantity or value of goods. You will receive compliance audits under this type of licence.
Help with choosing a licence
If you're unsure about which licence to select, then read the guidance on GOV.UK for exporting and doing business abroad.
